## Server & Client
서비스를 제공하고 사용하기 위해 네트워크에서 데이터를 주고받는다.

### 서버 클라이언트 구조
한 대의 서버에 다수의 클라이언트가 접속하여 서비스를 이용할 수 있는 방식

> Server

데이터를 주는 컴퓨터
- 서비스 제공

> Client

데이터를 요청하고 받는 컴퓨터
- 서비스 요청 및 사용

<br/>

## 애플리케이션

    프로그램, 솔루션, 시스템, 앱, 어플 등 다양한 지칭

컴퓨터의 역할을 결정
