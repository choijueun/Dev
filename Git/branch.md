# Branch

> git branch   

git branch 확인
- -a
- -r
- -v : 마지막 commit msg
- --merged: merged list
- --no-merged: no-merged list

   
> git branch (branch명)   

git branch 생성
- -c: 생성 후 branch 변경

   
> git switch (branch명)   

git branch 변경

   
> git push --set-upstream (remote명) (branch명)   

remote 명시
- Auto: git config --global push.default current

   
> git branch -d (branch명)   

git branch 삭제

   
> git branch -D (branch명)   

git branch 강제 삭제
